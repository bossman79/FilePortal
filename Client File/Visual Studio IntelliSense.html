IntelliSense in Visual Studio

IntelliSense is a code-completion aid that includes a number of features: List Members, Parameter Info, Quick Info, and Complete Word. These features help you to learn more about the code you're using, keep track of the parameters you're typing, and add calls to properties and methods with only a few keystrokes.

To enable intellisense for the Synergis Adept SDK, the following two files from the AdeptSDK\Client Files folder must be copied to the location where you will reference the Interop.AdeptCAC.dll and Interop.AdeptGui.dll files. This is usually the Client\PlugIn folder where Synergis Adept is installed.

  • Interop.AdeptCAC.xml
  • Interop.AdeptGui.xml
